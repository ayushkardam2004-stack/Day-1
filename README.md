# Day 1 MERN Training

This repository contains the projects completed during Day 1 of my MERN Stack training.

## Files Included

### 📄 index.html
- A portfolio webpage built using pure HTML.
- Demonstrates the use of semantic HTML tags for creating a well-structured and accessible layout.
- Includes sections such as header, about, skills, projects, and contact information.

### 🎉 collegeFest.html
- A College Fest webpage created using HTML.
- Showcases the use of forms, lists, links, and semantic tags.

## Technologies Used

- HTML5

## Learning Outcomes

- Understanding HTML document structure.
- Implementing semantic HTML tags.
- Creating responsive and organized web page layouts.
- Practicing forms and hyperlinks.
